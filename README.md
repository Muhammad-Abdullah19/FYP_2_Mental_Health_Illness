# Noor-e-Shifa (نورِ شفا)

**Bridging Science & Spirituality for Mental Wellness**

Noor-e-Shifa is an AI-powered platform designed to provide holistic mental health support by blending clinical guidance with authentic Islamic spiritual remedies. Built for accessibility, it offers a judgment-free space for emotional healing in both English and Urdu.

## 🚀 Key Features

- **🤖 AI Mental Health Assistant**: A 24/7 empathetic chatbot powered by Google's Gemini AI, offering immediate guidance and support.
- **📊 Clinical Assessments**: Standardized self-assessment tests for Depression (PHQ-9), Anxiety (GAD-7), and Stress (PSS-10).
- **🕌 Islamic Therapy**: A dedicated library of Quranic verses, Sunnah practices, and healing recitations tailored to emotional well-being.
- **📚 Disorder Library**: Comprehensive information on mental health conditions, symptoms, and coping mechanisms.
- **📈 Personal Dashboard**: Track your mental health journey, including mood stability and spiritual progress.
- **🌍 Bilingual Support**: Seamlessly switch between **English** and **Urdu (Phonetic)**.

## 💻 Tech Stack

- **Frontend**: React (Vite)
- **Styling**: Vanilla CSS (Custom Glassmorphism and Responsive Design)
- **AI Integration**: Google Gemini API
- **Icons & Fonts**: Font Awesome, Google Fonts (Inter, Poppins, Noto Nastaliq Urdu)
- **Routing**: React Router

## 🛠️ Getting Started

### Prerequisites
- Node.js (Latest LTS recommended)
- Google Gemini API Key

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-repo/FYP_1_Mental.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file in the root directory and add your API key:
   ```env
   REACT_APP_GEMINI_API_KEY=your_gemini_api_key_here
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## 📂 Project Structure

- `src/components`: Reusable UI elements (Navbar, AuthModal, Chatbot, etc.)
- `src/pages`: Main page components (Home, Assessment, Disorder, Islamic Therapy)
- `src/context`: Language and state management
- `src/data`: Global translations and static content
- `src/assets`: Images and media resources
